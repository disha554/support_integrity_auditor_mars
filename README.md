# Support Integrity Auditor

## Objective
Detect priority mismatches in customer support tickets.

## Dataset
Customer Support Tickets Dataset (20,000 tickets)

## Methodology
1. Data Preprocessing
2. Severity Signal Generation
3. Pseudo Label Generation
4. Random Forest Classification

## Features Used
- Ticket Text
- Resolution Time
- Satisfaction Score
- Keyword Severity

## Output
- Mismatch Label
- Hidden Crisis Detection
- False Alarm Detection

## Technologies
Python, Scikit-Learn, NLTK, Streamlit

## Future Work
Use DeBERTa and LLM-based evidence generation.
